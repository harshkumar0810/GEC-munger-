GEC Munger – UI Upgrade

Overview
This project is a complete UI redesign of the GEC Munger website. The primary objective was to create a modern, user-friendly interface with improved navigation, faster load times, and a clean layout tailored for students and faculty.

Tech Stack Used
* HTML5: For structuring the web pages.
* CSS3: For responsive and modern styling.
* JavaScript: For adding interactive UI elements.

Key Features
* Fully Responsive: Optimized layout that works seamlessly across mobile, tablet, and desktop devices.
* Improved Navigation: Restructured menus and flow to help users find information (like placements and academics) easily.
* Performance Optimized: Compressed assets and clean code structure for faster page loading.
